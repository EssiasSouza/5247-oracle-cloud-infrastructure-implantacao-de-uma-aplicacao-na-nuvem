# Oracle Cloud Infrastructure: implantação de uma aplicação na nuvem

Este repositório faz parte do curso da **Alura** e seu conteúdo é de propriedade da **Alura**.  
Ele foi preparado para auxiliar os alunos durante as aulas práticas do curso.

## Estrutura do projeto

O repositório contém dois arquivos principais:

- `./server1/index.html`
- `./server2/index.html`

Cada arquivo representa uma aplicação rodando em um servidor diferente.  
Quando utilizamos um **load balancer**, essas páginas são servidas de forma alternada, demonstrando o funcionamento de múltiplos servidores trabalhando juntos.

## Transparência para o usuário

Embora existam duas páginas distintas, para o usuário final a navegação deve parecer contínua e transparente.  
Algo foi propositalmente **escondido no código** para reforçar esse aprendizado. Esse detalhe será revelado e explicado durante a aula na Alura.

## Como usar este repositório

1. Faça um **fork** deste repositório para a sua conta do GitHub, **ou** copie os arquivos HTML diretamente para o ambiente que preferir.
2. Utilize os arquivos `index.html` de cada servidor (`server1` e `server2`) conforme as instruções que serão apresentadas na aula.

> ⚠️ Este repositório não ensina como criar ou configurar servidores na nuvem.  
> Esse processo completo será demonstrado passo a passo dentro do curso na plataforma da Alura.
